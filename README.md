# Introduction
Un GPS (Global Positioning System, en anglais) est un assistant de navigation ou assistant de navigation GPS, 
le plus souvent appelé GPS (par métonymie). C'est un appareil électronique portable ou embarqué dans un véhicule ; 
mais il peut aussi prendre la forme d'applications mobiles de navigation, disponibles sur smartphone. 
Il est capable de proposer des fonctions de navigation et de géolocalisation fournies par un système de 
positionnement par satellites (GNSS) dont le GPS américain, le Galileo européen ou le Glonass russe.
